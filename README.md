## 🎯 Learning Objectives
- Implement stack using arrays and linked lists
- Master Monotonic Stack for NGE/NSE problems
- Solve expression evaluation (infix/postfix/prefix)
- Implement Min Stack with O(1) getMin()
- Apply stack for backtracking and recursion simulation

## 📋 Problem Statements
### P1 — Next Greater Element: `[4,5,2,10]→[5,10,10,-1]`
### P2 — Valid Parentheses: `"()[]{}"→True`, `"(]"→False`
### P3 — Min Stack: push/pop/top/getMin all O(1)
### P4 — Stock Span: `[100,80,60,70,60,75,85]→[1,1,1,2,1,4,6]`
### P5 — Largest Rectangle Histogram: `[2,1,5,6,2,3]→10`
### P6 — Daily Temperatures: `[73,74,75,71,69,72,76,73]→[1,1,4,2,1,1,0,0]`
